h3. Liferay - Custom Field type "Date" for structured content

This hook enables portal moderators to create structured content with fields of type Date

h5. Component required:

	Liferay portal 6.1 CE ga1 

h5. Setup/Configure

	1. Download the war file at http://lplugins.download.rotterdam-cs.com, or get a copy from the GitHub repository.
	2. Put your war file in your <liferay-home>/deploy directory or add it through the Control Panel -> Plugins section
